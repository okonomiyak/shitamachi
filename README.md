# 下町 (Shitamachi)

日本の下町の風景をMinecraftで再現するMODです。
電柱・腕金・ケーブルなどを追加し、昭和レトロな街並みを作れます。

---

## 追加内容

### 電柱 (Utility Pole)
コンクリート製の電柱ブロック。腕金を取り付けたり、ケーブルで他の電柱と繋いだりできます。

### 腕金 (Cross Arm)
電柱に取り付けるアームブロック。電柱や他の腕金に隣接するだけで自動的に接続されます。

### ケーブル (Cable)
電柱同士を繋ぐアイテム。

1. 1本目の電柱を右クリック
2. 2本目の電柱を右クリック
3. ケーブルが張られます！

**Shift＋右クリック**でGUIを開き、ケーブルの**太さ**と**たわみ具合**をスライダーで調整できます。

### 警告カバー (Warning Cover)
電柱に取り付けられる警告マークのカバー。右クリックで付け外し、**Shift＋右クリック**で向きを変えられます。
